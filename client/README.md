The Fibonacci Client is a modern, high-performance web dashboard built with **Astro 5** and **React 19**. It provides a real-time visualization of Fibonacci calculations as they happen on the backend.

---

## üõ† Tech Stack

- **Framework**: [Astro 5](https://astro.build/) (utilizing the "Islands" architecture)
- **UI Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) & [DaisyUI](https://daisyui.com/)
- **State Management**: [Nanostores](https://github.com/nanostores/nanostores) (for lightweight, framework-agnostic state)
- **Real-time Engine**: 
  - **WebSockets**: [STOMP.js](https://stomp-js.github.io/stomp-websocket/) for bi-directional communication.
  - **SSE**: Native `EventSource` API for efficient server-to-client streaming.

- **Reactive Live Stream**: Watch Fibonacci numbers appear in real-time.

## üèó Architecture Note

This project leverages Astro's **Islands Architecture**. This means that most of the page is static HTML, while only the reactive components (like the live stream dashboard) are "hydrated" with React. This ensures lightning-fast initial load times while maintaining a rich, interactive experience.
