## üèó Module Architecture

The project is split into three specialized modules to ensure a clean separation of concerns:

- **[`core`](./core)**: 
    - Pure Java implementation of Fibonacci algorithms.
    - Zero external dependencies (no Spring).
    - Contains logic for Iterative, Recursive, and Fast Doubling approaches.
- **[`scheduler`](./scheduler)**: 
    - Manages background job execution using **JobRunr**.
    - Handles database persistence for calculation results.
    - Acts as the bridge between the computation logic and the data store.
- **[`web`](./web)**: 
    - The Spring Boot application entry point.
    - Provides REST endpoints for triggering calculations.
    - Implements **Server-Sent Events (SSE)** and **WebSockets (STOMP)** for real-time delivery.
    - Manages PostgreSQL `LISTEN/NOTIFY` for reactive updates.

## üöÄ Getting Started

### Prerequisites

- **Java 25**: Required for Virtual Threads and modern language features.
- **Maven 3.9+**: For building the project.
- **PostgreSQL**: Accessible via Docker or local installation.

### Building the Project

Run the following command from the `java` directory to build all modules and run tests:

```bash
./mvnw clean install
```

### Running the Application

To start the main web server:

```bash
./mvnw -pl web spring-boot:run
```

Alternatively, you can run the fat JAR:

```bash
java -jar web/target/web-1.0-SNAPSHOT.jar
```
